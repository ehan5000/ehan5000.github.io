Spring 2024

This AI Pathfinding Framework was developed using C++ and OpenGL. The project extends a maze creation and path planning demo by incorporating additional functionalities and enhancing the path planning algorithm to use the A* algorithm and the dijkstra's algorithm.

Key features of the project include:

  • Added a mobile sprite representing a character controlled by the mouse, with the character moving smoothly along the planned path.
  • Implemented visual indicators for nodes visited during path planning to display the search area.
  • Extended Dijkstra's algorithm to A* by adding an estimate of the remaining distance, using Manhattan distance multiplied by the lowest edge weight.
  • Modified the maze generation algorithm to use a random selection from a vector instead of a stack, affecting the maze structure.

Thorough documentation and comments were provided within the code. The project demonstrates advanced pathfinding techniques using both Dijkstra's and A* algorithms.

Please note that this project was developed for a school project. It is not intended for release and will not be released.
