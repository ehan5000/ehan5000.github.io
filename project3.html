Winter 2023

This playlist app was developed using JavaScript/NodeJS. The app allows users to create playlists, add and remove songs, and search for songs. It features a smart search function that suggests the best matches even when songs are misspelled. The app uses a unique login system and saves playlists for each account.

Through this project, I gained a deeper understanding of server-side development with NodeJS, including handling user authentication and managing sessions. I also enhanced my skills in creating and consuming RESTful APIs for dynamic web applications. Additionally, I learned about database management, utilizing both SQL and NoSQL databases to store and retrieve user data efficiently. Implementing the smart search functionality improved my knowledge of algorithms and data structures, particularly in the context of search optimization.

Please note that this app was developed for a school project. It is not intended for release and will not be released. Watch the demo video here: Playlist App Video.
