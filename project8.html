KuriusHacks Christmas Hackathon

Skills Used:

Created during the KuriusHacks Christmas Hackathon, this game is a festive twist on the classic “12 Days of Christmas” song, turned into an interactive, combat-driven 2D experience. Built using GML, C, and JavaScript, players navigate through progressively harder levels based on each verse of the song, facing enemies like "Three French Hens" and "Five Golden Rings" in themed combat zones.

The game was designed over five days under tight deadlines, emphasizing collaborative development, agile design, and thematic consistency. From sprite creation to event scripting and combat mechanics, everything was handcrafted for an immersive and joyful seasonal experience.

Players advance through colorful levels by defeating themed enemies, gathering resources, and timing their attacks. The gameplay uses an intuitive input system and features background music and sound effects tied to each “day” in the song.

Technologies Used

  • GML (GameMaker Language)
  • C
  • JavaScript
  • 2D Game Design
  • Hackathon Collaboration
