Winter 2021/2022

Created for the KuriusHacks: CE Christmas Hackathon, this project was developed using GML & JavaScript within a 5-day timeframe. The theme of the hackathon was "Christmas Song." Inspired by the song "12 Days of Christmas," each level features battles against characters from the song. For example, in level 3, players face off against "three French hens," and so on.

The game includes 2D Vector Projectile Motion, Collision Systems based on contact, Advanced AI Framework which can be used to edit/create many enemies, and a WIP Maze Generation which generates levels after you beat the game.

The game won 1st Place for Community Choice and 2nd Place for Jury Winner. You can play the game here: 12 Days of Christmas.
